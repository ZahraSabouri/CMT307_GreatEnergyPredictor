Important Evidence Limits

This report is evidence-based, but it should not be read as a perfect audit of every private conversation, meeting, Overleaf edit, or informal review.

Git confirms commits, authors, dates, file changes, and commit messages. It does not always prove who physically clicked "push" on GitHub, and it does not capture work done in Discord, meetings, Google Colab, Overleaf, Trello card discussions, or verbal coordination. Some commits are also GitHub web uploads or merge commits, so they can represent integration of someone else's notebook rather than original authorship of every line.

The final report was mainly produced in Overleaf. The downloaded CMT_307_Project.zip contains main.tex and report figures, and the PDF confirms the final report structure and model results, but those downloaded files do not contain Overleaf author history. Therefore, Overleaf-specific contributions are separated from Git-verifiable contributions where needed.

The spelling used in the repository is Shriya; this report uses that spelling even where meeting notes or spoken references used variants such as Shreya/Shria.

The Person 4 bundle is described here as a coordination and integration role: cross-cutting data integration, ensemble comparison, Sprint 3 review, report assembly, and final submission support. This is a university group project, so the wording below avoids corporate role labels and focuses on the actual tasks performed.
